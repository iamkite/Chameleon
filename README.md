<카멜레온>은 비전문가를 위한 VR 드로잉 애플리케이션이다. ‘쉽게 그리고 다르게 보세요!’라는 슬로건을 가지고 제작했다. <카멜레온>은 보다 더 많은 사람들이 즐길 수 있는 새로운 VR 모델링경험을 제안하고자, ‘양손 그리기’와 ‘렌더 모드 트랜지션’을 디자인하고 구현하였다.  
  
이곳에는 그리기와 편집 기능을 담당하였던 본인의 코드 부분을 올려두었다.
